📄 URD 使用者需求摘要

系統名稱：雲端檔案管理系統（Cloud File Manager）

系統目標：提供使用者管理雲端檔案的統一介面，支援多種檔案類型的瀏覽、搜尋、排序、標記與匯出。

  • ✅ 以樹狀結構呈現檔案與目錄的階層關係
  • ✅ 支援 Word 文件、圖片、純文字等多種檔案類型
  • ✅ 依副檔名搜尋檔案，即時顯示搜尋進度
  • ✅ 多種排序策略（名稱、大小、類型、標籤）
  • ✅ 檔案標記（Urgent / Work / Personal）
  • ✅ 刪除檔案，支援 Undo / Redo
  • ✅ 匯出 XML 格式與計算總容量

🧩 本系統使用的設計模式

Composite 檔案/目錄的樹狀結構
Visitor 走訪樹結構執行操作
Observer 搜尋事件即時通知
Command 操作封裝 + Undo/Redo
Strategy 可抽換的排序演算法
